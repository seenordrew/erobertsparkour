Quarry Bay Schedule

Mixed Class
(ages 13+)
Wednesdays
4:00pm - 5:30pm

This is a mixed age class for those that are young, and those that are young at heart. This is a light hearted, but hard working class where we will break-down tricky technical movements and difficult challenges in a welcoming environment. We will cover all the fundamental movements of parkour at a level appropriate for each practitioner.
 

Adults Beginner Class
(Ages 18+)
Tuesdays
7:00pm - 8:30pm

This adult class is a fun and playful way to get fit and learn the basics of parkour. It is a beginners class so anyone can jump right in. We will explore several different spots in and around Quarry Bay, using the environment to learn how to balance, crawl, vault, and climb with skill, while getting back in touch with our playful sides. This class is designed for complete beginners and is designed to help you get in shape, so there are no preprequisites to being able to join.
